# Algebra.ru ?

https://algebra-ru.azurewebsites.net

_Проект создан в целях получения опыта. При выборе идеи хотелось сделать продукт, который мог бы приносить пользу студентам. Основная идея -
посвятить весь материал одному предмету - алгебре (абстрактной и линейной)._

## Какой функционал доступен на данный момент ?

- Аутентификация
- Авторизация
- Добавления/удаления PDF файлов, постов, сервисов
- Смена личных данных в разделе "Контакты"

## Что использовалось при создании ?
- ASP .NET MVC

- EF Core
  > Будет заменено на Dapper
  
- Mongo DB (Grid FS)
  > Для хранения PDF файлов.
  
- Docker

- Bootstrap
